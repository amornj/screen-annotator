Privacy Policy

Effective date: February 12, 2026

Overview

Annotate is a screen annotation Chrome extension. Your privacy is important — this extension is designed to operate entirely on your device with no data collection.

Data Collection

Annotate does not collect, store, transmit, or share any personal data or browsing information.

Specifically, the extension:

  • Does not track your browsing history or activity
  • Does not collect any personal information
  • Does not send any data to external servers
  • Does not use analytics, cookies, or tracking technologies
  • Does not store any data beyond the current annotation session

Permissions

The extension requests the following Chrome permissions:

PermissionPurpose
activeTabAccess the current tab to overlay the annotation canvas and capture a screenshot when you choose to save
scriptingInject the annotation interface into the current page when you click the extension icon

These permissions are used only when you actively click the extension icon. No background data access occurs.

Screenshots

When you choose to save a screenshot:

  • The image is saved locally to your device (downloads folder)
  • The image is copied to your local clipboard
  • No image data is uploaded or transmitted anywhere

Third-Party Services

Annotate does not use any third-party services, SDKs, or libraries. It is built entirely with vanilla JavaScript with zero external dependencies.

Changes to This Policy

If this privacy policy is updated, the changes will be posted on this page with an updated effective date.

Contact

If you have questions about this privacy policy, please open an issue on the GitHub repository.
